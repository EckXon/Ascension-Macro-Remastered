# Ascension Macro Remastered
A Roblox Macro that checks for the presence of the specified user and, if they
are not in-game, rejoins the provided server link.

Steps:
1. Run the installer.bat file to install python and relevant packages
2. Run "python AMR.py" in the terminal (or double-click the file)
3. To quit, press F8. To force quit, press F9.
